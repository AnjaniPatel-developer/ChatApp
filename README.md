#  Chat Application

##  Project Description

This is a simple chat application built using Node.js, JavaScript, HTML, and CSS.
It allows users to communicate in real-time.
##  Features
* Real-time messaging
* Simple and clean UI
* Client-server communication

##  Technologies Used

* Node.js
* JavaScript
* HTML
* CSS

##  Project Structure

* `server.js` → Backend server
* `client.js` → Client-side logic
* `index.html` → Frontend UI
* `style.css` → Styling
* `package.json` → Project dependencies
